# Memory 模块

## 职责

Agent 的记忆系统——让 Agent 能记住过去的对话和用户偏好。

## 核心概念

```
MemoryManager
├── 短期记忆 (Session Memory)
│   └── 当前会话的对话历史
└── 长期记忆 (Long-term Memory)
    ├── 跨会话持久化
    └── 用户偏好（角色、风格、约束等）
```

## 设计思路

- **三级存储** — 短期（会话内）→ 长期（向量化）→ 用户偏好
- **编辑能力** — 用户可直接查看和编辑记忆
- **锁定机制** — 重要记忆可锁定，不被自动清理
- **临时关闭** — 可关闭特定记忆片段

## 后续规划

- [ ] 向量化存储（Embedding + 向量数据库）
- [ ] 记忆检索（相关记忆自动召回）
- [ ] 记忆加固（短期 → 长期转换策略）
- [ ] 用户偏好自动检测（从聊天中提取）
